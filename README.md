Вам нужно реализовать сайт о фильмах:

На главной странице должны отображаться карточки с фильмами (постер, название, дата выхода, страна).

Пользователь должен иметь возможность добавить свой фильм — есть отдельная страница с формой:

Название фильма

Описание фильма

Дата создания

Страна

Файл (постер)

При клике на фильм пользователь переходит на страницу, где содержится полная информация о фильме.
