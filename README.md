# Четыре задания: 

# 1

Вам нужно реализовать сайт о фильмах:

На главной странице должны отображаться карточки с фильмами (постер, название, дата выхода, страна).

Пользователь должен иметь возможность добавить свой фильм — есть отдельная страница с формой:

Название фильма

Описание фильма

Дата создания

Страна

Файл (постер)

При клике на фильм пользователь переходит на страницу, где содержится полная информация о фильме.

# 2

Данное задание основывается на предыдущем домашнем задании.
Создайте модель фильма.

1.Добавьте рейтинг от 1 до 5 (создайте ограничение для рейтинга от 1 до 5).
2. На странице должна быть возможность сортировать фильмы по:
- рейтингу
- дате создания
3. Обновите карточку фильма: добавьте отображение рейтинга. Обновите страницу фильма, добавив новые данные.
4. Добавьте возможность удалить или изменить фильм.

Загрузить ссылку на репозиторий проекта.

# 4

Продовжуємо роботу з кіно-сайтом.

1) Необхідно додати можливість управління відгуками:  
- Створити ReviewAdmin для зручного адміністрування відгуків  
- Налаштувати відображення імені користувача, тексту та дати створення відгуку  
- Додати фільтри за датою створення відгуку  
- Реалізувати можливість видалення та редагування відгуків через адмін-панель  

2) Додати управління жанрами:  
- Створити GenreAdmin для керування жанрами  
- Налаштувати можливість додавання, редагування та видалення жанрів  
- В MovieAdmin додати вибір жанрів через ManyToManyField  

3) Налаштувати відображення кількості відгуків до фільмів:  
- В MovieAdmin додати кастомне поле для підрахунку кількості відгуків  
- Використати list_display та list_filter для зручного пошуку  

4) Налаштувати права доступу:  
- Додати можливість призначати модераторів, які можуть керувати відгуками, але не змінювати фільми  
- Використовувати permissions у моделях Django  
- Створити групу "Модератори" та призначити їй право can_moderate_reviews  
- Додати перевірку прав у ReviewAdmin, щоб модератори могли редагувати та видаляти лише відгуки  
- Обмежити зміну фільмів, дозволивши це лише суперкористувачам  

5) Кастомізація адмін-панелі:  
- Використовувати list_display, search_fields, ordering та list_filter для покращення функціональності  
- Додати кастомні стилі для адмін-панелі  
- Використовувати спадкування шаблонів для єдиного оформлення сайту  
- Використати статичні файли для стилізації адмін-панелі
