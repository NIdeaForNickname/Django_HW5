# Три задания: 

# 1

Вам нужно реализовать сайт о фильмах:

На главной странице должны отображаться карточки с фильмами (постер, название, дата выхода, страна).

Пользователь должен иметь возможность добавить свой фильм — есть отдельная страница с формой:

Название фильма

Описание фильма

Дата создания

Страна

Файл (постер)

При клике на фильм пользователь переходит на страницу, где содержится полная информация о фильме.

# 2

Данное задание основывается на предыдущем домашнем задании.
Создайте модель фильма.

1.Добавьте рейтинг от 1 до 5 (создайте ограничение для рейтинга от 1 до 5).
2. На странице должна быть возможность сортировать фильмы по:
- рейтингу
- дате создания
3. Обновите карточку фильма: добавьте отображение рейтинга. Обновите страницу фильма, добавив новые данные.
4. Добавьте возможность удалить или изменить фильм.

Загрузить ссылку на репозиторий проекта.
